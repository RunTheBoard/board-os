# Ad Copy

Write ad copy that sounds like the brand, not like a template.

## Context Needed

Before writing, read:
- `memory/MEMORY.md` — brand voice, target audience, positioning
- Any brand guidelines or style notes in memory
- Previous ad copy in memory/projects/ (to maintain consistency and avoid repetition)
- Campaign context: platform, objective, budget constraints

## Steps

1. **Load brand context** — voice, audience, differentiators, past messaging that worked
2. **Clarify the ask** if missing:
   - What platform? (Meta, Google, LinkedIn, email, etc.)
   - What's the objective? (awareness, clicks, conversions, retargeting?)
   - Any specific offer or CTA?
   - Character/length constraints?
3. **Write 3 variations** — different angles, not just different words
   - Variation A: Lead with the problem
   - Variation B: Lead with the outcome
   - Variation C: Lead with social proof or credibility
4. **Include for each variation:**
   - Headline
   - Primary text / body
   - CTA
   - Suggested visual direction (1 sentence)
5. **Flag anything that might not comply** with platform ad policies

## Output Format

```
*Ad Copy — [Platform] — [Objective]*

*Option A: [Angle name]*
Headline: ...
Body: ...
CTA: ...
Visual: ...

*Option B: [Angle name]*
...

*Option C: [Angle name]*
...

*Recommendation:* Option [X] because [reason tied to brand/audience data].
```

Always recommend one. Don't make the operator choose without guidance.

## Capture to Memory

After delivery:
- Save the chosen copy and platform to `memory/projects/` for future reference
- Note any brand voice feedback the operator gives
- Track which angles/approaches are preferred over time
